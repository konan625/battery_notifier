# Battery Notifier

A lightweight desktop battery monitoring utility built in Rust to explore systems programming and native OS notifications.

The application continuously tracks battery percentage and alerts the user when charging thresholds are reached, helping prevent overcharging and unnecessary battery drain.

## Features

- Real-time battery monitoring
- Desktop notifications for charging thresholds
- Lightweight and minimal resource usage
- Built while learning Rust and systems-level programming concepts

## Tech Stack

- Rust

## Goal

This project was built primarily to get hands-on experience with Rust, native system interactions, and lightweight background utilities.
